# Student Task Manager

A simple and user-friendly web application to manage daily tasks efficiently.
This project allows users to add, view, update, and delete tasks, along with
a side dashboard that displays all tasks in real time.

## Features
- Add new tasks
- Delete existing tasks
- Mark tasks as completed
- Real-time task updates
- Side dashboard showing all added tasks

## Tech Stack
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## Project Structure
Student-Task-Manager/
- index.html
- style.css
- script.js
- README.md

## How It Works
- Users can enter a task using the input field
- Tasks are displayed in the main task list
- The side dashboard updates automatically whenever a task is added or removed

## Purpose
This project was built to strengthen front-end development skills and
demonstrate basic CRUD operations using JavaScript.
